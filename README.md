# Bufete Extranjería España

Sitio web profesional para un bufete de abogados especializado en la regularización de inmigrantes en España bajo la nueva ley de extranjería.

## Descripción

Este proyecto es una página web construida con Next.js para un bufete de abogados que se enfoca en ayudar a clientes con procesos de inmigración y regularización en España. Incluye secciones para servicios, información sobre el bufete y un formulario de contacto.

## Tecnologías Utilizadas

- **Next.js**: Framework de React para aplicaciones web.
- **TypeScript**: Lenguaje de programación tipado.
- **Tailwind CSS**: Framework de CSS para estilos.
- **ESLint**: Herramienta de linting para JavaScript/TypeScript.

## Instalación

1. Clona el repositorio:
   ```bash
   git clone <url-del-repositorio>
   cd extranjeria
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```

4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Construcción para Producción

Para construir la aplicación para producción:

```bash
npm run build
```

## Despliegue

La aplicación puede ser desplegada en plataformas como Vercel, Netlify o cualquier servidor que soporte Next.js.
